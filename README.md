# Yonzon Autogears

This project is implemented using Vite React as a front end framework and Redux for state management.

## Installation

To install the project, follow these steps:
1. Clone the repository: `git clone <repository_url>`
2. Navigate to the project directory: `cd autogear-retail-management`
3. Install dependencies: `npm install`
4. Start the development server: `npm run dev`

## Pages

1. **Auth**: Contains login functionality.
2. **Dashboard**: Displays sales data and charts.
3. **Inventory**: Allows adding, updating, deleting, and searching stocks.
4. **Invoice**: Handles transactions.
5. **Settings**: Lists access settings.

## User Access

There are two types of users with specific access:
- **Owner (1)**: Has full access to all pages.
- **Cashier (2)**: Limited access to certain pages.

## Deployment

The front end is deployed using Vercel.

## Purpose

This project was primarily developed as a capstone project for a retail autogear shop.
